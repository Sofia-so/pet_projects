Web Application on Flask 

Технології: Python, Flask, SQLAlchemy, PostgreSQL, Flask-Login, Jinja2, HTML, Git. 

• Реалізовано реєстрацію та авторизацію користувачів за допомогою Flask-Login. 

• Розроблено функціонал керування обліковими записами (редагування та видалення профілю). 

• Реалізовано роботу з базою даних PostgreSQL через SQLAlchemy ORM. 

• Налаштовано валідацію даних форм та обробку помилок користувацького вводу. 

• Створено HTML-шаблони з використанням Jinja2. 

• Використано Git для контролю версій та керування кодовою базою.
